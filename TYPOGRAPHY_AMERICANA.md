## Contrast & Readability Analysis

### Navy Headings on White Background

```
Color Contrast:
Navy Blue (#0f172a) on White (#FFFFFF)
Contrast Ratio: 16.8:1

WCAG Rating: AAA (Excellent)
Minimum Required: 4.5:1 for normal text
Minimum Required: 3:1 for large text (headings)

Result: Exceptional readability, professional appearance
```

### Charcoal Body Text on White

```
Color Contrast:
Charcoal (#4A4A4A) on White (#FFFFFF)
Contrast Ratio: 9.2:1

WCAG Rating: AAA (Excellent)
Minimum Required: 4.5:1 for normal text

Result: Perfect for long-form reading, no eye strain
```

### Why This Hierarchy Works

1. **Navy headings:** Darker color = visual weight = importance
2. **Charcoal body:** Slightly lighter = subordinate = supporting info
3. **Clear distinction:** Eye immediately knows what's a heading vs. body text

## Technical Implementation

### CSS Custom Properties

```css
:root {
    /* Americana Palette */
    --color-navy: #0f172a;           /* Headings, Trust */
    --color-text-secondary: #4A4A4A; /* Body, Readability */
    --color-brand: #A82020;          /* CTAs, Action */
}

/* Global Heading Styles */
h1, h2, h3, h4, h5, h6 {
    font-family: 'Oswald', sans-serif;
    color: var(--color-navy);
    font-weight: 800;
    text-transform: uppercase;
    letter-spacing: -0.02em;
}

/* Body Text */
body {
    font-family: 'Source Sans 3', sans-serif;
    color: var(--color-text-secondary);
    font-size: 18px;
    line-height: 1.7;
}
```

## Exceptions & Overrides

### When to Use White Text (Not Navy)

1. **Dark Backgrounds:**
   - Quote form section (charcoal background)
   - Footer (navy background)
   - Any overlay text

2. **Hero Section:**
   - Hero headline uses white for contrast against navy overlay
   - Exception to the rule for maximum impact

### Override Example:

```html
<!-- Default: Navy heading -->
<h2>SERVICES</h2>

<!-- Dark background: White heading -->
<section class="bg-charcoal">
    <h2 class="text-white">GET YOUR QUOTE</h2>
</section>
```
