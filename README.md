- **Border Radius:** `0px` (STRICT - enforced via CSS `!important`)
- **Container:** `1600px` max-width (extra wide breakout)

- Semantic HTML5 (`<article>`, `<section>`, `<nav>`, `<header>`, `<footer>`)
- ARIA labels on all interactive elements
- `aria-hidden="true"` on decorative icons
- High contrast text (WCAG AA compliant)
- Keyboard navigation support
- Focus-visible states on all interactive elements
- Alt text on all images

#### Performance Optimizations
- Native lazy loading on below-the-fold images (`loading="lazy"`)
- Tailwind CSS via CDN (production build)
- GPU-accelerated animations (`transform: translateZ(0)`)
- Optimized Google Fonts loading (preconnect)
- Intersection Observer for image loading fallback

## üìù CUSTOMIZATION GUIDE

### Updating Colors
Edit `css/style.css` root variables:
```css
:root {
    --color-brand: #A82020;      /* Primary red */
    --color-charcoal: #1F2937;   /* Secondary dark */
}
```

### Changing Fonts
Update Google Fonts link in `index.html`:
```html
<link href="https://fonts.googleapis.com/css2?family=YourFont...">
```

### Adding New Sections
1. Copy existing section structure from `index.html`
2. Maintain `style="padding-top: 80px; padding-bottom: 80px;"`
3. Ensure `id` attribute for navigation linking
4. Use semantic HTML tags (`<section>`, `<article>`)
